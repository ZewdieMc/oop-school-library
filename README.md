# 📖 OOP school library <a name="about-project"></a>

A OOP school library app is a software application that can be used to manage a school library. The app can be used to track the inventory of books. The app can also be used to track the students who have borrowed books from the library.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

  <ul>
    <li><a href="https://www.ruby-lang.org/en/">Ruby</a></li>
  </ul>

### Key Features <a name="key-features"></a>

- Add Teacher
- Add Student
- Add Book
- Borrow Book

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

<ul>
    <li><a href="https://www.ruby-lang.org/en/">Ruby</a></li>
  </ul>
  
### Setup

Clone this repository to your desired folder:

```sh
cd my-folder
git clone https://github.com/ZewdieMc/oop-school-library.git

```


### Install

Install this project with:

```sh
cd oop-school-library
bundle install
```


### Usage

To run the project, execute the following command:

```sh
ruby file_name.rb
```
